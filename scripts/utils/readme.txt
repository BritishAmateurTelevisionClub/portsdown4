The bash scripts in this folder can be used to recompile and install updates to 
some of the Portsdown c programs:

uguir.sh     compiles and runs the scheduler and hence runs rpidatvtouch.c
ugui.sh      compiles rpidatvtouch.c
guir.sh      runs the scheduler and hence rpidatvtouch.c

usgr.sh      compiles and runs siggentouch.c

udvb.sh       compiles libdvbmod, compiles DvbTsToIQ and then copies dvb2iq to the bin folder

ulst.sh       compiles LimeSDR Toolbox and copies the 4 binaries to the bin folder
